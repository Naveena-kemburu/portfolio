# Kemburu Naveena — Personal Portfolio

A fully responsive personal portfolio website built with React, Vite, Tailwind CSS, and CSS scroll animations.


## Tech Stack

- React 19 + Vite
- Tailwind CSS v3
- CSS IntersectionObserver (on-scroll animations)
- Vanilla JS parallax (hero background)
- @iconify/react (icons)

## Live Site

[https://your-name.vercel.app](https://your-name.vercel.app)

## Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
